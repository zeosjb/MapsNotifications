# Visualización Interactiva del Mapa y Notificaciones

## 📱 Características

- 🗺️ **Mapa interactivo** con ubicación del usuario y movimiento para que el pueda llegar a destino.
- 🔔 **Notificaciones push** para mostrar actualizaciones.

## 🛠️ Tecnologías

- React Native
- React Navigation
- Expo Location y Maps
- Expo Notifications

## 🚀 Instalación

1. Clona el repositorio:

   ```
     git clone https://github.com/zeosjb/MapsNotifications.git
   ```
2. Instala dependencias:
   ```
     npm install
   ```
3. Crea un archivo .env con tus claves:
   ```
    GOOGLE_MAPS_API_KEY=tu_api_key
   ```
4. Iniciar el proyecto:
   ```
    npx expo start
   ```
